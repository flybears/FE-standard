# CSS开发规范

### 1 命名

#### [强制] 组成元素

* 命名必须由单词、中划线或数字组成。
  
* 不允许使用拼音（约定俗成的除外，如：youku,baidu），尤其是缩写的拼音、拼音和英文混合。
  
  示例：
  
  ``` css
  /** good **/
  .detail{
    display: none;
  }
  .news-list{
    display: block;
  }
  
  /** bad **/
  .xiangqing{
    display: none;
  }
  .new_list{
    display: block;
  }
  ```
